# Language Length Analysis

Compare how text expands and contracts across languages — a linguistics and localization analysis tool.

**[Live Demo](https://language-length-analysis.generaltranslation.dev)** | **[General Translation Docs](https://generaltranslation.com/docs)**

## About

This app visualizes how translated text changes in length across languages. It shows average expansion ratios, side-by-side character and byte comparisons, and interactive examples — helping developers understand the real-world impact of internationalization on UI layouts.

## GT Features Used

- `<T>` — JSX translation
- `<Num>` — Number formatting
- `<Var>` — Dynamic values in translations
- `getGT` — Server-side string translations
- `<LocaleSelector>` — Language picker
- `loadTranslations` — Local translation storage

## Getting Started

```bash
git clone https://github.com/gt-examples/language-length-analysis.git
cd language-length-analysis
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
